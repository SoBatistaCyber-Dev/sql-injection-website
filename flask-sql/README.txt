To install packages:

    pip install Flask Flask-Scss Flask-MySQLdb

To upgrade pip:

    pip install --upgrade pip

To freeze requirements:

    pip freeze > requirements.txt

To list all packages in the current env:

    pip list

To create a new env:

    python3 -m venv env

To activate the env:

    source env/bin/activate

To run the app:

    python3 app.py
